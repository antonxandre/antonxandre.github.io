## Destaque de Resiliência Técnica e Arquitetura

**Case: Interoperabilidade Legado-Mobile (Firebird & Flutter)**
Capacidade comprovada de projetar soluções de infraestrutura para contornar gargalos de arquitetura. Destaque para o desenvolvimento de um aplicativo Flutter integrado a um SGBD Firebird de ERP legado, onde as respostas trafegavam estritamente em XML. 
*   **A Solução:** Configuração e implantação de uma Máquina Virtual (Windows Server) atuando como *middleware*.
*   **O Pipeline:** Desenvolvimento de um fluxo de serialização complexo, interceptando os dados do ERP, realizando a conversão `XML > JSON`, processando os objetos complexos no Dart, convertendo novamente para JSON e submetendo a carga validada para serviços externos.
